# [永夜出版社](https://idleditor.fivood.com) · Idle Editor

> 一个不死吸血鬼编辑的日常。审稿、出版、赚声望、熬死伯爵——挂机为辅，经营为主。

你是一位在永夜出版社工作了 217 年的吸血鬼。出版社的茶水间里没有镜子、红茶配方可疑、实习生被一代又一代换掉——但你从来没变过。

## 核心玩法

稿件不断涌入投稿池。每份稿件都有书名、类型和一段随机生成的简介。**你的工作是判断：审稿、退稿还是搁置？**

- 新来稿默认**折叠遮盖**，点「翻阅」触发翻页进度条动画，进度满后展开简介 + 封面缩略图 + 操作按钮
- 退掉烂稿赚修点和声望。退掉好书扣声望，**签约作者的稿子退掉会减好感**，好书作者有概率被**竞争出版社永久挖走**
- 审稿后进入编辑流水线：审稿 → 编辑(可选精校) → 校对 → 封面查看 → 出版
- 出版赚修点、声望和版税。烂书出版扣声望。**每月有出版额度限制**（可用版税扩展）
- 搁置的书 5-10 分钟后作者修改重投（品质 +3，简介追加修改说明）

|------|------|----------|
| T1 | Lv.3 | 速读(+20%翻阅) / 快笔(+15%编辑) / 温和批评(+10%好感) |

## 主要系统

### 作者
9 种人格（退休教授、地下室宅、拉美大师、北欧推理女王、日本轻小说宅等），英文名配中文吐槽译名。三段晋升(签约→知名→传奇)。隐藏好感度——出版积好感、退稿扣好感、满值成为忠实作者(+5才华)。**点击作者卡片弹出详情面板**：人格类型+口头禅、好感度进度条、出版历史、互动按钮(请吃饭+15好感/催稿减冷却减好感)。

### 伯爵叙事
每次转生不只是数字变化。**第 1/3/5/7 次转生触发专属剧情场景**，玩家选择影响与伯爵的关系值。第 5 次转生可选择伯爵性别。**第 10 次转生根据关系值触发三种不同结局**。

### 竞争出版社
五大对手——晨曦、新纪元、万象、冷泉、星云——各有专门领域和人格化描述。对手社的动态出现在编辑部日志中，包括挖角签约作者、推出竞品版、书展交锋等。

### 随机决策
每 ~10 分钟弹出一个二选一决策事件。LLM 根据游戏状态实时生成（失败回退 12 个模板）。**选择后果实打实影响游戏状态**——RP/声望/品质/作者冷却自动解析应用。

### AI 集成
- 审稿通过/退稿自动生成编辑吐槽（KV 缓存、多人共用、每书每场景最多 5 条轮换）
- 随机决策事件实时生成
- 转生时 LLM 撰写本世编辑生涯总结
- 300 条预生成简介池（6 类型 × 50 条 × 4 文体）

## 图书与封面
232 个标题 × 6 类型。69 篇手写戏仿简介。封面优先 PNG → 无则 SVG → 都没则占位图标。

## 书房（内置阅读器）
上传 .txt / .md / .epub。EPUB 自动解析章节。**左右分栏**——左侧正文右侧简介+书签+书架列表。阅读进度每 5 秒自动保存，手动书签。切换标签页不丢状态。
