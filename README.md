# [永夜出版社](https://idleditor.fivood.com) · Idle Editor

> 一个活了 217 年的吸血鬼编辑的日常。审稿、出版、养猫、熬死伯爵——挂机为辅，经营为主。

你是一位不朽的吸血鬼。永夜出版社（你的同类创办）聘请你来当编辑，他们觉得反正你闲着也是闲着。出版社的茶水间里没有镜子、红茶配方可疑、实习生被一代又一代换掉，但你从来没变过。

## 核心玩法

稿件不断涌入投稿池。每份稿件按类型分色稿纸样式——**你的工作是判断：审稿、退稿还是搁置？**

- 新来稿默认**折叠遮盖**，点「翻阅」触发翻页进度条动画，进度满后展开简介 + 操作按钮
- 稿件卡片显示**初筛印象**标签（初筛存疑→潜力之作四档），不翻阅也能快速判断
- 退掉烂稿赚修点和声望。退掉好书扣声望，签约作者退稿减好感，有概率被**竞争出版社永久挖走**
- 审稿后进入编辑流水线：审稿 → 编辑(可选精校) → 校对 → 封面查看 → 出版
- 投稿池上限 **7 份**，闲置超 10 分钟的稿子自动消失
- **征稿系统**：公开征稿（免费）/ 定向约稿（30 RP，品质+10）/ 加急征稿（200 版税）三选一

## 作者系统

**17 种人格**——退休教授、地下室科幻宅、前情报官员、社会学博士、焦虑新人、隐居拉美作家、北欧推理女王、美国畅销流水线、日本轻小说宅、考据派历史作家、奇幻史诗执笔、法国文学隐者、印度史诗圣者、俄式沉重长卷、韩国网文女王、尼日利亚魔幻写实、澳洲内陆哥特。

- **生涯上限**：高产型（韩国网文女王 >40 本）→ 一书封神型（印度史诗圣者 1-3 本），封笔后不再投稿
- 三段晋升(签约→知名→传奇) + 隐藏好感度 + 忠诚标记
- 互动：请吃饭/寄样书/写回信（120s 共用冷却），好感数值卡片显示
- **解约**功能——弹窗确认："婚姻都有破裂的可能…" 我后悔了 / 祝你幸福

## 猫生系统

收养猫为**随机决策事件**（窗外来了一只猫）。取名（最多6字）、摸猫（+3好感，60s冷却）、逐年变老。好感 ≥ 80 且猫龄 ≥ 5 时可**消耗 1 铜像赐予永生**。未永生且好感 < 50 的猫在 10 岁后可能离世。6 条猫专属事件 + 12 条超自然西幻随机事件（女巫、狼人、龙等）。

## 图书与封面

**150+ 个标题** × 6 类型。**95+ 篇手写戏仿简介**——涵盖科幻经典（三体/沙丘/神经漫游者）、推理名作、悬疑、社科经典以及**奇幻史诗戏仿**（时光之轮、冰与火之歌、魔戒、猎魔人、碟形世界等），每本书有原著剧情戏仿描述。非戏仿书统一使用「主角」而非角色名。投稿池为按类型分色的稿纸样式（蓝/紫/红/黄/绿/粉）。书架全白色书脊。

## 天赋树

编辑等级提升解锁天赋。**5 层 15 个天赋，每层三选一，永久生效，纪元保留。**

## 纪元系统（原"转生"）

满足条件后顶栏出现「纪元」按钮。铸造铜像重置进度，保留永久加成。**第 1/3/5/7 次纪元触发伯爵剧情场景**，第 5 次可选伯爵性别，第 10 次触发结局。纪元后给启动资金（100 RP + 500 版税）。弹窗展示本世回顾（出版数/退稿数/作者数/等级）。办公室可删档重开。

## 编辑批语

- 出版时自动生成模板批语存入书籍
- 封面确认弹窗可**手动提交批语**（提交后锁死，出版后才能在书架编辑）
- 书架弹窗「写批语」按钮调 LLM 生成，**单次使用后隐藏**
- 批语操作写入编辑日志：`{玩家名}飞快写下了对《xxx》的批语：…`

## 部门系统

4 个部门（编辑/设计/市场/版权），**升级成本随等级递增**（×1.5），进度条实时显示。升级完成 toast 确认。设计部在封面阶段加品质，版权部每 tick 产出声望。**审稿门槛滑块**可设品质下限，低于门槛的稿件全自动出版。

## 界面

- **像素 CSS 背景**：桌面端砖墙 + 哥特窗 + 木桌三层纹理，移动端噪点纹理
- **日志面板**：最新在上，按年月分组，细线分割 + 小字标签
- **退稿/审稿完成提示**：12+15 种模板轮换，根据玩家性别切换 她/他

## 书房（内置阅读器）

上传 .txt / .md / .epub。EPUB 自动解析章节。阅读进度每 5 秒自动保存，手动书签。切换标签页不丢状态。
