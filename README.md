# [永夜出版社](https://idleditor.fivood.com) · Idle Editor

> 一个不死编辑的日常。你是一位在永夜出版社工作了 217 年的吸血鬼编辑——审稿、出版、赚声望、熬死老板。

## 核心玩法

稿件不断涌入。每份稿件都有书名、类型和一段随机生成的简介。你的工作是判断：**审稿、退稿还是搁置？**

- 新来稿默认折叠，需要点「翻阅」才能看到简介——揭开谜底感
- 退掉烂稿赚修点和声望；退掉好书扣声望，作者可能被**竞争出版社挖走**
- 审稿后进入编辑流水线：审稿 → 编辑 → 校对 → 封面 → 出版
- 出版赚修点、声望和版税。烂书出版扣声望

## 主要系统

### 投稿与自动化
每个 tick 都可能产生新稿件。编辑部 Lv.3 解锁自动审稿，声望 100 解锁自动出版，声望 200 解锁自动退稿。每个特权有独立 ON/OFF 开关。搁置的书 5-10 分钟后作者修改重投，品质 +3。

### 作者系统
9 种人格（退休教授、地下室宅、拉美大师、北欧推理女王、日本轻小说宅等），英文名配中文吐槽译名。签约 → 知名 → 传奇三段晋升。隐藏好感度系统——好感高了写感谢信，满值成为忠实作者。点击作者卡片弹出详情：人格描述、口头禅、好感度进度条、出版历史、互动按钮（请吃饭 +15 好感 / 催稿减冷却）。

### 经济循环
修点（RP）通过出版/退稿赚取，用于部门升级、精校、作者互动。声望（声）出版和退掉烂稿获得。版税（税）被动收入——在办公室消费：扩展出版额度、雇佣公关、装修阅览室、赞助文学奖。

### 精校
编辑阶段花 RP 提升稿件品质：轻校 (10RP/+3Q)、深校 (30RP/+8Q)、极校 (60RP/+15Q)。

### 书籍再版
已出版书可花版税再版：品质 +3、进入 7 天营销窗口期（销量 ×1.5）。

### 收藏成就
6 个类型里程碑（推理 5 本→销量+5%、科幻 5 本→品质+3% 等），首次解锁时给永久加成。

### 随机决策
每 ~10 分钟弹出一个二选一事件。LLM 根据游戏状态实时生成，失败回退 12 个模板。每个选项有明确数值后果。

### 编辑等级
出版和退稿积累 XP，TopBar 显示 Lv.X 进度条。离线每分钟 +1 XP。

### 竞争出版社
五大对手——晨曦、新纪元、万象、冷泉、星云——各有专门领域和人格化描述。退稿作者可能被挖走，对手社的动态出现在编辑部日志中。

### 伯爵叙事
转生不只是数字变化。每 1/3/5/7 次转生触发专属剧情场景，玩家选择影响与伯爵的关系值。第 5 次转生可选择伯爵性别。第 10 次转生触发结局——根据关系值有三种不同终点。

## 图书与封面

### 标题库
232 个唯一标题，6 个类型。69 篇手写戏仿简介（"第22条军规"、"洛丽塔争议版"等）。300 条 LLM 预生成池 + 模板后备。4 种文体（叙述/摘录/书评/腰封）。

### 封面系统
有 PNG 显示 PNG，没有显示 SVG 几何占位。185 种配色和纹理的 SVG 封面自动生成。

### 书架
已出版书籍以书脊视图展示，支持排序/筛选。统计栏显示总销量和最畅销书名。

## 书房（内置阅读器）

上传小说——支持 .txt / .md / .epub。EPUB 自动解析章节。左右分栏：左侧正文右侧简介+书签+书架列表。阅读进度每 5 秒自动保存，手动书签命名和跳转。切换标签页不丢失阅读状态。

## AI 集成

- 审稿/退稿自动生成编辑吐槽（KV 缓存，多人共用，每书每场景最多 5 条轮换）
- 随机决策事件实时生成
- 300 条预生成简介池（6 类型 × 50 条）
- 全部走服务端 Key，用户无需配置

## 技术栈

React 19 + TypeScript + Vite + Tailwind v4 · Zustand · Dexie.js (IndexedDB) · Cloudflare Pages + Functions + KV · JSZip (EPUB)
