# 永夜出版社 · Idle Editor

> [idleditor.fivood.com](https://idleditor.fivood.com) — 一个不死编辑的日常。

你是一位在永夜出版社工作了 217 年的吸血鬼编辑。审稿、出版、赚声望、熬死老板——挂机为辅，经营为主。

## 核心玩法

你面对不断涌入的投稿。每份稿件都有书名、类型和一段随机生成的简介。你的工作是判断：**审稿还是退稿？**

- 退掉烂稿赚修点和声望；退掉好书扣声望。
- 审稿后进入编辑流水线：审稿 → 编辑 → 校对 → 封面 → 出版。
- 出版赚修点、声望和版税。烂书出版扣声望。
- **盲判**：投稿不显示品质分数和退稿理由——你只能根据简介做判断。

随着游戏推进，玩家从手动审稿逐步过渡到经营整个出版社。

## 主要系统

### 部门与自动化
四个部门可雇佣和升级：编辑（加速流水线）、设计（提升封面品质）、市场（增加销量）、版权（被动声望）。编辑部 Lv.3 解锁自动审稿，声望达标后解锁自动出版和自动退稿。

### 作者系统
9 种人格（退休教授、地下室宅、拉美大师、北欧推理女王、日本轻小说宅等），签约后自动投稿。声望系统支持三段晋升（签约→知名→传奇）。隐藏好感度系统——好感高了写感谢信，满值成为忠实作者。

### 经济循环
修点（RP）通过出版/退稿赚取，用于部门升级和精校。声望（声）出版和退掉烂稿获得，门槛解锁特权和话语权。版税（税）被动收入——在办公室消费**扩展每月出版额度**（100/200/400...）。

### 精校
编辑阶段花 RP 提升稿件品质：轻校 (10RP/+3Q)、深校 (30RP/+8Q)、极校 (60RP/+15Q)。每份稿限一次——值得花 RP 的书毕竟不多。

### 随机决策
每 ~10 分钟弹出一个二选一事件：书评人想提前审读、影视改编报价、作者预支稿费…… LLM 根据游戏状态实时生成，失败回退 12 个模板。每个选项有明确的数值后果。

### 编辑等级
出版和退稿积累 XP，TopBar 显示 Lv.X 进度条。离线每分钟 +1 XP（在线约 20 倍效率）。等级影响品质、速度和收入（预留，即将启用）。

### 转生与终极目标
累计一本畅销书后解锁转生。重置进度但保留铜像永久加成（品质/速度/版税/才华）。每转生一次消耗吸血鬼伯爵 1 年寿命——归零即通关。

## 图书与封面

### 标题库
185 个唯一标题，涵盖科幻、推理、悬疑、社科、混合、轻小说 6 个类型。42 部现实文学戏仿（"第22条军规"、"绿楼梦"、"洛丽塔（争议版）"等）拥有手写简介。

### 封面系统
每本书自动匹配封面：有 PNG 显示 PNG，没有显示 SVG 几何占位。104 种不同配色和纹理的 SVG 封面自动生成。玩家可自行添加 PNG 到 `public/covers/`。

### 书架
已出版书籍以**书脊视图**展示（截取封面左侧），点击弹出详情弹窗：200×280 封面 + 简介 + 编辑批语。

## 书房（内置阅读器）

底部导航第五个标签。上传自己的小说——支持 **.txt / .md / .epub** 格式。自动存阅读进度，手动书签，简洁排版。Markdown 渲染标题/加粗/斜体/代码/列表。EPUB 自动解析章节结构去标签。切换标签页不丢失阅读状态。

数据纯本地存储（IndexedDB），不上传服务器。

## AI 集成

三个系统接入 LLM（DeepSeek，服务端 Key）：
- 投稿卡片 **AI 简介重写**按钮
- 随机决策事件的**实时生成**
- 书架详情 **AI 编辑评语**按钮

每人每会话限制 30 次调用。不需要用户配置 API Key。

## 技术栈

React 19 + TypeScript + Vite + Tailwind v4 · Zustand · Dexie.js (IndexedDB) · Cloudflare Pages + Functions + KV · JSZip (EPUB)

## 本地开发

```bash
npm install
npm run dev       # Vite 前端
npm run build     # 生产构建
```

## 封面生成脚本

```bash
node scripts/gen-svg-covers.mjs          # 生成 SVG 封面
node scripts/gen-covers-gemini.mjs       # LLM 生成英文提示词（需 .env 中填 Key）
```

## 环境变量（Cloudflare Pages）

| Variable | 说明 |
|----------|------|
| `LLM_API_KEY` | DeepSeek/OpenAI API Key |
| `LLM_BASE_URL` | 默认 `https://api.deepseek.com` |
| `LLM_MODEL` | 默认 `deepseek-chat` |
